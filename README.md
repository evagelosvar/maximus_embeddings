# maximus_embeddings
Embeddings of 1100 chapters of Maximus the confessor  based on  BAAI/bge-m3 model, in original Byzantine Greek and in two Modern Greek translations. The texts and embeddings are aligned. The translations can be used only for research issues.

- data_gr1.csv -> Galitis translation and embeddings
- data_gr2.csv ->Meretakis translation and embeddings
- data_grc-> Original byzantine text and embeddings
